# Procure-to-Pay (P2P) Performance Dashboard

## 📊 Overview
This project analyzes the Procure-to-Pay (P2P) process using Power BI. It focuses on invoice lifecycle, payment efficiency, and vendor performance.

## 🎯 Objectives
- Track invoice processing efficiency
- Identify delays in payment cycle
- Analyze vendor performance
- Improve financial decision-making

## 📌 Key Metrics
- Total Invoices
- On-Time %
- Late %
- Pending %
- Avg Delay (Days)
- Total Amount

## 🛠 Tools Used
- Power BI
- Excel

## 💡 Insights
- Certain vendors contribute more to delays
- A portion of invoices remain pending
- Payment trends vary across months

This project demonstrates skills in financial analysis, data visualization, and process improvement.
